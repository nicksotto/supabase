Schedule regularly running background jobs with the Cron Jobs integration.

## About

Under the hood, the Cron Jobs Module uses the [`pg_cron`](https://github.com/citusdata/pg_cron) Postgres extension.

<Admonition type="note">

`pg_cron` is not fully supported on Fly Postgres. Learn more about [Fly Postgres limitations](/docs/guides/platform/fly-postgres#limitations).

</Admonition>
